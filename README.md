# tello-tempo
Tello Tempo makes Tello drone dance


Commands

| What        | Fingers     | Keyboard       |
|-------------|-------------|----------------|
| play music  | 1           | b              |
| stop music  | 2           | n              |
| dance swing | 3           | c              |
| stop swing  | 4           | v              |
| stop & quit |             | q              |
| volume set  | index/thumb |                |
| takeoff     |             | &lt;tab>       |
| land        |             | &lt;backspace> |
| move left   |             | left arrow     |
| move right  |             | right arrow    |
| move up     |             | top arrow      |
| move down   |             | low arrow      |
| stop move   |             | x              |
